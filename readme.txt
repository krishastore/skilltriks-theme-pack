Transform your WordPress site into a dynamic Learning Management System (LMS) using our innovative WordPress LMS Plugin.

== Description ==

**Extend the visual power of your SkillTriks LMS with professionally designed and customizable themes!**

The **SkillTriks Theme Pack** is an official add-on for the **<a href="https://www.skilltriks.com/" target="_blank">SkillTriks LMS plugin</a>** that brings a collection of beautifully crafted themes to elevate the look and feel of your e-learning website.

Each theme in the pack is fully responsive, easy to apply, and comes with customization options that let you tweak colors, fonts, and visual styles without touching a single line of code. Whether you're an educator, course creator, or organization using SkillTriks, this add-on gives you the flexibility to align your LMS appearance with your brand.

**New themes and customization features will be added regularly!**

== Features ==

* Multiple theme options for your SkillTriks-powered LMS
* Customize fonts and colors to match your brand
* One-click theme activation
* Fully responsive design
* No coding required
* Regular updates with new themes
* Seamless integration with SkillTriks core plugin

== Installation ==

1. Make sure the **<a href="https://www.skilltriks.com/" target="_blank">SkillTriks LMS plugin</a>** is installed and activated.
2. Upload the skilltriks-theme-pack plugin folder to the /wp-content/plugins/directory, or install the plugin through the WordPress plugin.
3. Activate the plugin through the ‘Plugins’ screen in WordPress.
4. Go to the SkillTriks > Settings page to select and apply your preferred theme.

== Frequently Asked Questions ==

**1. Is this a standalone plugin?**  
     No. This is an add-on and requires the main **<a href="https://www.skilltriks.com/" target="_blank">SkillTriks LMS plugin</a>** to function.

**2. Can I customize fonts and colors?**  
     Yes! Each theme allows you to customize **fonts, primary/secondary colors, button styles,** and more—directly from the admin settings..

**3. Can I switch themes anytime?**  
     Yes! You can change themes with a single click from the Theme Settings page.

**4. Will more themes be added in the future?**  
     Absolutely. We’re working on adding more unique, user-friendly themes to the pack regularly.

**5. Will my content or courses be affected if I switch themes?**  
     Not at all. Your course content, user data, and progress remain untouched. Only the visual layout changes.
